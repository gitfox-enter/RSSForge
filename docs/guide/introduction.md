# 什么是 RSSForge

RSSForge 是一款**基于 GitHub Actions 的免费 RSS 订阅源生成器**。

## 解决的问题

很多网站不提供 RSS 订阅，但你又不想每次打开网站刷新闻。RSSForge 就是来解决这个问题的：

- 你想订阅一个线报站，但网站没有 RSS
- 你想追踪一个论坛的新帖子，但网站没有通知功能
- 你想监控一个产品页面的价格变化

**RSSForge 全部搞定**。

## 工作原理

```
你配置想订阅的网站 → GitHub Actions 自动定时抓取 → 生成 RSS 文件 → 你用阅读器订阅
```

整个流程在你的 GitHub 仓库里运行，不占用任何服务器资源，不花钱。

## 与 RSSHub 的区别

| | RSSHub | RSSForge |
|---|---|---|
| 部署方式 | 自建服务器 / Docker | Fork + GitHub Pages |
| 成本 | 需要服务器 | 完全免费 |
| 订阅方式 | 每次访问时实时生成 | 预先生成静态 XML |
| 适合场景 | 临时查阅、少量路由 | 长期稳定订阅 |
| 定制难度 | 需要写路由规则 | 修改 sites.yaml 即可 |

## 技术特性

- **多解析器支持**：内置 40+ 网站专用解析器，自动识别站点类型
- **JS 渲染支持**：Playwright 抓取需要 JavaScript 渲染的页面
- **智能调度**：按站点等级（high/mid/low）自动分配抓取频率，夜间自动降频
- **自动去重**：基于内容相似度去重，不会出现重复条目
- **历史回填**：支持分页抓取，首次添加站点时拉取历史内容
- **站点图标**：每个订阅源自动显示对应网站的 favicon 图标
